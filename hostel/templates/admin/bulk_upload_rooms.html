Tips
  • Any number of category rows per floor — mix and match variants freely.
  • Student room counts must sum to End No. − Start No. + 1. Mismatch skips the floor.
  • Common rooms (Gym, Mess, etc.) each get their own floor row with Floor=GF, Start=End.
  • Room numbers assigned row by row: 4S+CWR first, then 3S+WR+AC, then 2S+CWR, etc.
  • Beds are auto-created for student rooms only (a 4S+CWR+AC room gets 4 beds, etc.).
  • New hostels are created automatically if they don't already exist.
  • Duplicate room numbers in the same hostel are safely skipped with a note.
  • Use GF as the floor label for Ground Floor (maps to floor 0).
