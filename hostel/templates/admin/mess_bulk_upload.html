How Monthly Updates Work
  1. Upload Excel → system creates one entry per day per meal.
  2. Each day's menu is served automatically when that date arrives.
  3. If you manually edit a day's menu (from the Menu page), it becomes "protected".
  4. Next month, upload again → protected entries are skipped (your edit is preserved).
  5. To overwrite a protected entry, use Force Replace after uploading.
  6. Upload the same month again with a new file to update any unprotected day.
Tips
  • Each hostel has its own menu — upload once per hostel, or select "All Hostels".
  • Leave a cell blank to show "Not set" for that meal.
  • Column order doesn't matter — the system detects by header name.
  • Students and mess incharge see today's daily menu automatically.
  • If no daily menu is found for today, the weekly default menu is shown as fallback.
