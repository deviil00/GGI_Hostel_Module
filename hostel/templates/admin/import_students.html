How It Works
  1. Upload Excel → student profiles created instantly and visible to student, admin & superadmin.
  2. If Email (D) is provided → login account auto-created (temp password = Roll Number).
  3. Superadmin/Admin fills Hostel (P) + Room (Q) in the sheet or via the student profile → room allocation auto-assigned.
  4. Student profile shows room, floor, bed number as soon as allocation is saved.
  5. Upload the same Roll Number again → record is updated, not duplicated.
Tips
  • Hostel Name must match exactly as entered in the system (case-insensitive).
  • Room No. must already exist in that hostel — create rooms first via Bulk Upload Rooms.
  • Guardian Phone (M) is used for gate pass & leave approval — always include it.
  • DOB format is DD-MM-YYYY, e.g. 15-03-2005.
  • Semester defaults to 1 if left blank; Year defaults to 1.
  • Rows with missing Roll Number or Name are automatically skipped.
